Step By Step: Manually Set up Resource Groups for initial deployment of the web application

Creating a resource group

  1. Open the Azure Portal by navigating to https://portal.azure.com
  2. Click "Create a resource"

  1. In the search box, search for "Resource Group"

  1. In the search results, click "Resource group" and then click "Create"

  1. Enter the resource group name, select your subscription and select the location closest to you as Resource group location. Then click "Create".

  1. After a little while your resource group will be created. Click "Go to resource group" to open it.

  1. You will see that your freshly created resource group is empty.

Create security group and add members to security group

  1. In the Azure portal navigate to the Azure Active Directory blade

  1. In the blade, select Groups

  1. Press the + button to create a new security group
  2. Create a group call Azure (teamname). Select as Group type "Security"
  3. Select Members in the same blade

  1. Once the group is created, select "Members"
  2. Press the + Add Members button
  3. Search for "g-venue-" and select the members from your team and save the group

Assign security group to resource group

  1. Find your Resource Group e.g. use the Search box
  2. In your resource group, click "Access control (IAM)", click "Add", select "Contributor" for the role and find your team through the search box.

  1. Select your team and click "Save". You should see your team being added as "Contributor" specific for this resource.
